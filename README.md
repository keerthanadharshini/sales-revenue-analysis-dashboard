# Sales Revenue Analysis Dashboard

## Overview

This project was developed as part of my Data Science Internship at Thiranex.

The dashboard analyzes sales and revenue data and provides interactive insights through visualizations and KPI tracking.

## Features

* Total Revenue KPI
* Total Quantity Sold KPI
* Revenue Trend Analysis
* Top Performing Products
* Interactive Dashboard

## Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## Dataset

The project uses a sample sales dataset containing product, category, quantity, price, and revenue information.

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
